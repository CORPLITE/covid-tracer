# covid-tracer
COVIDTracer QRCode based Contactless Alternative to Pen & Paper Check-ins.

[Information Site](https://covidtracer.org.au/)

[Live Site](https://covidtracer.corplite.com/)

## Installation
```bash
yarn install
```

## Usage
Development Server
```bash
yarn run dev
```
